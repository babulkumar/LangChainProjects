ABOUT PROJECT
=============
This is a premlinary project to understand and implement a Chat Bot

Steps to execute
================
1. Create API key in OPENAI.
2. Create new virtual environment - Use python -m venv .venv 
 a. Activate this virtual environment - .venv\Scripts\activate
3. Activate the virtual environment - Use conda activate venv/
4. Create a file requirements.txt in the project. This file has following list of softwares:-
 a. langchain
 b. OPENAI
 c. huggingface_hub
5. Next pip install -r requirements.txt
6. pip install ipykernel ==> This is not required in requirements.txt
7. Then start with langchain.ipynb 
8. Need .env file to write down the OPENAI API keys. 
9. Create a .gitignore file and add .env to it so that it is never uploaded to Github.
10. Use python-dotenv library
