ABOUT PROJECT
=============
This is a premlinary project to understand and implement a Chat Bot

Steps to execute
================
1. Create API key in OPENAI.
2. Create new virtual environment - Use conda create -p venv python==3.9 -y
3. Activate the virtual environment - Use conda activate venv/
4. Create a file requirements.txt in the project. This file has following list of softwares:-
 a. langchain
 b. OPENAI
 c. huggingfacehub
5. Next pip install -r requirements.txt
